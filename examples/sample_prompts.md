## Tips for Writing Effective Prompts

1. **Be Specific**: Include details about requirements, constraints, and expected behavior
2. **Mention Technology**: Specify libraries, frameworks, or approaches to use
3. **Include Context**: Reference existing code or patterns when relevant
4. **State Goals**: Explain what you're trying to achieve
5. **Add Examples**: Provide examples of inputs/outputs when helpful

## Example Prompt Structure

```
[Action] [What] [How] [Additional Requirements]

Example:
Create a user authentication system using JWT tokens with email verification and password reset functionality
```
